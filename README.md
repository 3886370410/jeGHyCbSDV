# 微商城后台管理系统 java199

## 项目概述

微商城后台管理系统是一个基于 Vue、Node.js 和 Mysql 技术栈构建的综合性管理平台。该系统主要包括用户登录、商品管理、分类管理以及个人信息管理等核心功能模块。

## 技术栈

- 前端：Vue.js
- 后端：Node.js
- 数据库：Mysql

## 功能模块

- 用户登录
- 商品管理
  - 添加商品
  - 修改商品
  - 删除商品
  - 商品列表
- 分类管理
  - 添加分类
  - 修改分类
  - 删除分类
  - 分类列表
- 个人信息管理
  - 修改密码
  - 更新个人信息

## 系统角色

- 管理员：负责系统的管理与维护

## 运行环境

- 前端：支持现代浏览器
- 后端：Node.js 环境搭建
- 数据库：Mysql 数据库

## 部署步骤

1. 数据库搭建与配置
   - 安装 Mysql 数据库
   - 导入数据库结构
   - 配置数据库连接
2. 后端服务部署
   - 安装 Node.js
   - 克隆或下载项目代码
   - 安装依赖
   - 配置相关环境变量
   - 运行 Node.js 服务器
3. 前端界面部署
   - 克隆或下载前端项目代码
   - 安装依赖
   - 运行前端构建
   - 部署至 Web 服务器

## 核心亮点

- 前后端分离，便于维护和扩展
- 采用Node.js，实现高性能的 RESTful API
- 支持用户权限管理，保证系统安全可靠
- 灵活的数据库设计，易于后续功能扩展和优化
